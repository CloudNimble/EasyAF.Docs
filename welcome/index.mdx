# Welcome to EasyAF

EasyAF is a comprehensive .NET framework ecosystem designed to accelerate application development with a focus on Blazor, messaging, testing, and RESTful services.

## What is EasyAF?

EasyAF brings together multiple powerful libraries and tools that work seamlessly together:

- **Core Framework** - Foundational libraries providing base classes, utilities, and abstractions
- **BlazorEssentials** - Essential Blazor components and utilities
- **Breakdance** - Testing and debugging utilities
- **SimpleMessageBus** - Lightweight message-based communication
- **RESTier** - OData-based REST API framework

## Why EasyAF?

### Accelerated Development
Build applications faster with pre-built components, utilities, and patterns.

### Seamless Integration
All components are designed to work together out of the box.

### Battle-Tested
Used in production by numerous applications across various industries.

### Modern Architecture
Built on the latest .NET technologies with a focus on performance and maintainability.
