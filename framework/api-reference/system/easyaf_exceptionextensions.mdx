### TraceDemystifiedException

```csharp
public static Exception TraceDemystifiedException(this Exception ex, string logPrefix = "")
```

Demystifies the Exception and writes it to <xref href="System.Diagnostics.Trace.TraceError(System.String%2cSystem.Object%5b%5d)" data-throw-if-not-resolved="false"></xref>.

#### Parameters

- **ex** [Exception](https://learn.microsoft.com/dotnet/api/system.exception): The exception instance to manipulate.
- **logPrefix** [string](https://learn.microsoft.com/dotnet/api/system.string): A string that will be prepended to the log entry. Defaults to the calling function name.
