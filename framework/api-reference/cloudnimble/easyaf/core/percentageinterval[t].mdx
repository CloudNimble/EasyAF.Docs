# PercentageInterval&lt;T&gt;

<Badge variant="outline">Class</Badge>

Represents a percentage rate that occurs at regular time intervals, enabling conversion between different time periods.
This class combines a base time interval (from the <xref href="CloudNimble.EasyAF.Core.Interval%601" data-throw-if-not-resolved="false"></xref> class) with a percentage rate to calculate 
total percentage amounts across different time periods.

## Syntax

```csharp
public class PercentageInterval<T> : Interval<T> where T : IComparable<T>, IConvertible
```

## Remarks

<p>
<strong>Key Concepts:</strong>
</p>
<ul><li><strong>Value:</strong> Always represents a time duration (e.g., 3 hours, 1.5 days)</li><li><strong>Type:</strong> The time unit for the Value (Hours, Days, Months, etc.)</li><li><strong>Rate:</strong> The decimal percentage rate that occurs each interval period</li></ul>

<p>
<strong>Method Types:</strong>
</p>
<ul><li><strong>Per* methods (inherited):</strong> Calculate how many intervals fit in a time period</li><li><strong>RatePer* methods:</strong> Calculate total percentage rate for a time period (intervals Ã— rate)</li></ul>

<p>
<strong>Common Use Cases:</strong>
</p>
<ul><li>Interest rates: "Earn 2.5% interest every quarter"</li><li>Growth rates: "Achieve 5% growth every month"</li><li>Error rates: "Allow maximum 0.1% error rate every hour"</li><li>Discount rates: "Apply 10% discount every week"</li></ul>
