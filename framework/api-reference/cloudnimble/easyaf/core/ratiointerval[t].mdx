# RatioInterval&lt;T&gt;

<Badge variant="outline">Class</Badge>

Represents a ratio value that occurs at regular time intervals, enabling conversion between different time periods.
This class combines a base time interval (from the <xref href="CloudNimble.EasyAF.Core.Interval%601" data-throw-if-not-resolved="false"></xref> class) with a ratio value to calculate 
total ratio amounts across different time periods.

## Syntax

```csharp
public class RatioInterval<T> : Interval<T> where T : IComparable<T>, IConvertible
```

## Remarks

<p>
<strong>Key Concepts:</strong>
</p>
<ul><li><strong>Value:</strong> Always represents a time duration (e.g., 1.5 hours, 2 days)</li><li><strong>Type:</strong> The time unit for the Value (Hours, Days, Months, etc.)</li><li><strong>Ratio:</strong> The decimal ratio that occurs each interval period</li></ul>

<p>
<strong>Method Types:</strong>
</p>
<ul><li><strong>Per* methods (inherited):</strong> Calculate how many intervals fit in a time period</li><li><strong>RatioPer* methods:</strong> Calculate total ratio value for a time period (intervals Ã— ratio)</li></ul>

<p>
<strong>Common Use Cases:</strong>
</p>
<ul><li>Conversion rates: "Convert 70% of leads every 2 weeks"</li><li>Performance metrics: "Achieve 0.95 efficiency ratio every 8 hours"</li><li>Quality metrics: "Maintain 0.99 success ratio every day"</li></ul>
