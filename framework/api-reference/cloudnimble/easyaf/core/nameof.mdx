# NameOf

<Badge variant="outline">Class</Badge>

Fills a gap in <a href="https://learn.microsoft.com/dotnet/csharp/language-reference/operators/nameof">nameof</a> by allowing you to use deep name references instead of local name references.

## Methods

### Full&lt;TSource&gt;

```csharp
public static string Full<TSource>(Expression<Func<TSource, object>> expression, string separator = ".")
```

Gets the full property path name from the specified expression, optionally using a custom separator.

#### Parameters

- **expression** [Expression&lt;Func&lt;TSource, object&gt;&gt;](https://learn.microsoft.com/dotnet/api/system.linq.expressions.expression-1): An expression pointing to the property whose full name should be returned.
- **separator** [string](https://learn.microsoft.com/dotnet/api/system.string): The character(s) used to separate property names in the result. Defaults to ".".

#### Returns

[string](https://learn.microsoft.com/dotnet/api/system.string)

The full property path as a string with the specified separator.
