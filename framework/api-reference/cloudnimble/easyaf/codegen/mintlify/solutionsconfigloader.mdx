# SolutionsConfigLoader

<Badge variant="outline">Class</Badge>

Loads and validates solutions.json configuration files for multi-solution documentation generation.


### LoadAsync

```csharp

```

Loads and validates a solutions.json configuration file.

#### Parameters

- **filePath** [string](https://learn.microsoft.com/dotnet/api/system.string): The path to the solutions.json file.

#### Returns

[Task&lt;SolutionsConfig&gt;](https://learn.microsoft.com/dotnet/api/system.threading.tasks.task-1)

A validated SolutionsConfig object.

#### Exceptions

- `[ArgumentNullException](https://learn.microsoft.com/dotnet/api/system.argumentnullexception)`: Thrown when filePath is null or whitespace.
- `[FileNotFoundException](https://learn.microsoft.com/dotnet/api/system.io.filenotfoundexception)`: Thrown when the solutions.json file does not exist.
- `[InvalidOperationException](https://learn.microsoft.com/dotnet/api/system.invalidoperationexception)`: Thrown when the configuration is invalid.

- **config** [SolutionsConfig](/api-reference/cloudnimble/easyaf/codegen/mintlify/models/solutionsconfig): The solutions configuration containing custom ignore patterns.

#### Returns

[bool](https://learn.microsoft.com/dotnet/api/system.boolean)

## Remarks

This class handles parsing of solutions.json files, applies default configurations,
and validates the resulting configuration for multi-solution documentation generation.
