# XmlDocumentationProcessor

<Badge variant="outline">Class</Badge>

Processes XML documentation elements and applies link resolution.

```csharp
public static class XmlDocumentationProcessor
```

## Inheritance

## Methods

### ProcessElement

```csharp
public static string ProcessElement(XmlDocumentationElement element, MintlifyLinkResolver linkResolver)
```

Processes an XML documentation element and resolves any cross-references.

#### Parameters

- **element** [XmlDocumentationElement](/api-reference/cloudnimble/easyaf/xmldocumentation/xmldocumentationelement): The documentation element to process.
- **linkResolver** [MintlifyLinkResolver](/api-reference/cloudnimble/easyaf/codegen/mintlify/mintlifylinkresolver): The link resolver to use for cross-references.

#### Returns

[string](https://learn.microsoft.com/dotnet/api/system.string)

The processed MDX content with resolved links.

## Remarks

This class provides a centralized way to process XML documentation elements
and apply cross-reference link resolution using the MintlifyLinkResolver.
It acts as a bridge between the parsed XML elements and the final MDX output.
