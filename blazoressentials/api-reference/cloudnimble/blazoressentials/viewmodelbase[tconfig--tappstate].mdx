# ViewModelBase&lt;TConfig, TAppState&gt;

<Badge variant="outline">Class</Badge>

A base class for your Blazor MVVM implementation that gives you access to all the useful stuff Blazor and BlazorEssentials inject into the app.

## Syntax

```csharp
public class ViewModelBase<TConfig, TAppState> : BlazorObservable, INotifyPropertyChanged, IDisposable where TConfig : ConfigurationBase where TAppState : AppStateBase
```
