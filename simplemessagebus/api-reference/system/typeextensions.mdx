## Methods

### SimpleAssemblyQualifiedName

```csharp
public static string SimpleAssemblyQualifiedName(this Type type)
```

Guarantees the creation of an AssemblyQualifiedName that does not contain version or key details. That way when AssemblyVersions are incremented,
the system will still attempt to process the <xref href="CloudNimble.SimpleMessageBus.Core.IMessage" data-throw-if-not-resolved="false"></xref>.

#### Parameters

- **type** [Type](https://learn.microsoft.com/dotnet/api/system.type): 

#### Returns

[string](https://learn.microsoft.com/dotnet/api/system.string)

A string containing the <code class="paramref">type</code> name in the format "FullTypeName, SimpleAssemblyName".
