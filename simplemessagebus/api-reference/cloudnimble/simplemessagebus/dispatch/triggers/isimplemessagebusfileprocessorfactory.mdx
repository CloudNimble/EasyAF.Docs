# ISimpleMessageBusFileProcessorFactory

<Badge variant="outline">Interface</Badge>

Factory interface for creating <xref href="CloudNimble.SimpleMessageBus.Dispatch.Triggers.SimpleMessageBusFileProcessor" data-throw-if-not-resolved="false"></xref> instances. This factory pattern allows
different FileProcessors to be used for different job functions.

## Syntax

```csharp
public interface ISimpleMessageBusFileProcessorFactory
```

## Methods

### CreateFileProcessor

```csharp

```
